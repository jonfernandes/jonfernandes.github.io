Can I book a ticket to travel to one city but return from another city?

It'sÂ not possible to book a return ticket to travel to one city but return from another city. For example if you book a return ticket to go to Paris you would not be able to return from Brussels. Your ticket would only be valid to travel from Paris.

In this situation you would need to book two single journey tickets one to Paris and one from Brussels. To do this via the web site simply tick the box to book a 'One way' journey.
